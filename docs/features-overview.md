# ClaudeTask 功能概览

## 系统架构

ClaudeTask 采用现代化的前后端分离架构，提供稳定、高效的 AI 任务执行平台。

### 技术栈
- **前端**：React + Vite + Ant Design
- **后端**：Flask + SQLAlchemy + WebSocket
- **数据库**：SQLite（轻量级）/ PostgreSQL（生产环境）
- **AI 引擎**：Claude API / Local LLM
- **版本控制**：Git + GitHub API
